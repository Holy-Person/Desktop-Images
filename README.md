# Desktop-Images

[![GitHub last commit](https://img.shields.io/github/last-commit/Holy-Person/Desktop-Images?style=for-the-badge)](https://github.com/Holy-Person/Desktop-Images)
[![GitHub code size in bytes](https://img.shields.io/github/languages/code-size/Holy-Person/Desktop-Images?style=for-the-badge)](https://github.com/Holy-Person/Desktop-Images)
[![GitHub](https://img.shields.io/github/license/Holy-Person/Desktop-Images?style=for-the-badge)](https://github.com/Holy-Person/Desktop-Images)

**Desktop Images** lets you display images on your desktop, just open the app and drag any image onto the window, the window can be minimized without affecting the individual images.
Click and drag images around and use the various keybinds indicated on the window to scale, flip, duplicate or delete individual images.

Made using Godot 4.1.4.

Support for .gif files powered by BOTLANNER's [Godot GIF](https://github.com/BOTLANNER/godot-gif).
